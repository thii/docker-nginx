docker-nginx
============

Blazing fast nginx with latest stable version of nginx, memcached, PHP and HHVM preinstalled.

## Usage
```bash
$ docker pull thii/nginx
```

## Deploying to AWS Elastic Beanstalk
```bash
$ eb init
$ eb start
$ git aws.push
```
